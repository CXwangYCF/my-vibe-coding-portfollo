两个作品分别对应行业场景和教育场景。

Project 01

行业文档理解与方案生成助手

面向公安、消防、政企等材料密集型场景，把非结构化文档整理为需求拆解、 PRD 草稿、汇报提纲与资源计划。这个作品更偏 AI 产品经理视角， 强调如何把模型能力包装成一套可演示、可讲述、可延展的产品工作台。

  • 目标用户：产品经理、售前顾问、项目协同角色
  • 功能结构：材料输入、需求分析、PRD 输出、汇报导出
  • 展示重点：多页面结构、工作台交互、模型切换与 API 接入预留

Project 02

数学题错题诊断助手

面向初中数学学习场景的错题复盘网页，通过题目、解题过程和答案输入， 提供错因定位、知识点讲解、复习建议与同类练习，强调诊断式反馈。

  • 目标用户：初中生、家长、辅导场景
  • 功能结构：错因分析、复习路径、练习反馈、错题记录
  • 展示重点：教育产品语言、用户陪伴感和学习闭环设计
