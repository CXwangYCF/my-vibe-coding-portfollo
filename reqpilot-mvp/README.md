# ReqPilot MVP

`ReqPilot` 是一个面向政企 / 公安 / 消防场景的 `行业文档理解与方案生成助手`。

当前版本是一个可直接打开演示的前端 MVP，重点验证以下能力：

- 上传或粘贴行业材料
- 自动输出需求拆解卡片
- 自动生成 PRD 草稿
- 自动生成客户汇报提纲
- 展示产品指标、资源计划和模型选型

## 当前已实现

- 3 个内置样例场景：公安警情复盘、消防巡检项目、政企招标材料
- 工作台式交互：输入区 + 结果区 + 多标签页
- 可编辑输出：PRD 草稿和汇报提纲支持直接修改
- MVP 级规则分析：根据文本关键词生成角色、痛点、功能、风险与问题

## 适合作品集展示的讲法

1. 目标用户是谁
2. 用户为什么不想再手工整理材料
3. MVP 为什么先做需求拆解、PRD、汇报提纲
4. 这个工作台如何帮助 PM / 售前 / 项目助理提升效率
5. 后续如何接入 OCR、知识库和正式大模型调用
